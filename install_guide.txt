Для развертывания в среде Arch/Manjaro/Endeavour выполняются команды
 sudo pacman -Syu
 sudo pacman -S base-devel cmake gcc opencv blas lapack git python-pip
 pip install -r requirements.txt

 Для развертывания в среде Ubuntu/Debian выполняются команды
 sudo apt update && sudo apt upgrade -y
 sudo apt install -y build-essential cmake libopenblas-dev liblapack-dev libx11-dev libgtk-3-dev python3-pip
 pip install -r requirements.txt

Для развертывания в среде Windows требуется установить
 Visual Studio C++ Build Tools
 CMake с добавлением в системный PATH
выполнить
  pip install -r requirements.txt

Спецификация файла requirements.txt включает в себя пакеты 
 numpy, opencv-python, face_recognition, scipy и pillow

Для запуска системы выполняется команда
 python main_dashboard.py.

----------| ENGLISH VERSION |----------

For deployment in an Arch/Manjaro/Endeavour environment, execute the following commands
 sudo pacman -Syu
 sudo pacman -S base-devel cmake gcc opencv blas lapack git python-pip
 pip install -r requirements.txt

For deployment in an Ubuntu/Debian environment, execute the following commands
 sudo apt update && sudo apt upgrade -y
 sudo apt install -y build-essential cmake libopenblas-dev liblapack-dev libx11-dev libgtk-3-dev python3-pip
 pip install -r requirements.txt

For deployment in a Windows environment, it is required to install
 Visual Studio C++ Build Tools
 CMake with system PATH configuration enabled
execute the following command
 pip install -r requirements.txt

The requirements.txt file specification includes the following packages
 numpy opencv-python face_recognition scipy pillow

To launch the system, execute the following command
 python main_dashboard.py
